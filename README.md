# Flag Tag

A multiplayer keep away game with a flag for Decentraland!

**Five minute rounds running 24/7**

## Description
Multiplayer keep away game where players compete to hold the flag for the longest time. Built with Decentraland SDK7.

## Deployment
- **World**: flagtag.dcl.eth
- **Scene Size**: 160m x 240m (150 parcels)

## Development

### Prerequisites
- Node.js 18+
- Decentraland SDK 7

### Local Development
```bash
npm install
npm run start
```

### Deploy
```bash
npm run deploy
```
